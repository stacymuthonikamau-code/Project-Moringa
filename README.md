## Description

-Project-Moringa is a responsive photography and portfolio website designed to showcase images, personal information, and contact details in a clean and simple layout.
-The project focuses on practicing multi-page website structure using pure HTML and CSS, creating a visually appealing gallery and navigation system without using heavy frameworks.

## Features
-Responsive multi-page layout
-Image gallery for showcasing photos
-Navigation between pages (Home, About, Gallery, Contact)
-Clean and simple design for easy browsing

## Learning Goals
-Practice structuring multi-page websites
-Improve CSS layout and styling skills
-Learn image handling in web projects
-Understand basic UI design principles
-Use GitHub for version control
